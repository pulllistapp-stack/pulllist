PullList는 **포켓몬 TCG 진지한 컬렉터**를 위한 카탈로그 + 가격 추적 도구입니다. 일본판 / 미국판 / 한국판을 한 곳에서, 매일 갱신되는 시세와 함께.

## 어떤 기능이 있나요

- **31,000+ 카드** — 영문 / 일본판 / 한국판 (TCGdex + pokemontcg.io)
- **매일 갱신되는 시세** — eBay 실시간 listing + TCGCSV 일일 스냅샷
- **트렌딩** — 7일 / 30일 / 90일 / 1년 가격 변동, Bulk / Chase 티어 분리
- **포트폴리오** — 보유 카드 + 가치 추적 + 시간별 그래프
- **위시리스트** — 목표 가격 설정 + 임계값 도달 알림
- **카드 검색** — 영문/일본어 cross-language. "Charizard" 검색하면 "リザードン" 카드도 잡힘

## 시작하는 법

1. **[카드 둘러보기](/cards)** — 31,000+ 카드 그리드 필터링
2. **[Sets](/sets)** — 셋별 정리, 컬렉션 완성도 확인
3. **[Trending](/trending)** — 지금 시세 오르는 카드 모음
4. **로그인 → 포트폴리오** — 보유 카드 등록하고 자산 변화 추적

## 일판 카탈로그가 강점

기존 미국판 중심 도구와 다르게 PullList는 **일본판 = 1급 시민**입니다.

- 일본판 셋 170+개 인덱싱 완료
- 일본 한정 프로모 (JPP-XY, JPP-SwSh, JPP-SM, JPP-SV) 별도 분류
- 일본판 희귀도 표기 (RR / SR / SAR / AR / UR / HR / SSR ...)
- 일본판 ↔ 미국판 동일 카드 cross-language 검색

## 앞으로

- 일본판 시세 별도 통합 (현재 미국 시세 위주)
- 한국판 셋 자동 인덱싱
- 컬렉션 export / 공개 포트폴리오 공유
- 카드별 가격 알림 (이메일 + 푸시)
